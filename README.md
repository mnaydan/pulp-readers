# pulp-readers
This repository contains in-progress files and notebooks for a research and development project led by Dr. Mary Naydan to identify and parse readers' letters from early twentieth-century pulp science fiction and fantasy magazines. 

Bulk downloaded PDFs of the collections in Internet Archive available here:
- [Astounding Stories Collection](https://drive.google.com/drive/folders/17JNihP_5J9y8cEane5dlaIw7WOfRfUse?usp=drive_link)
- [Amazing Stories Collection](https://drive.google.com/drive/folders/1kE_ZdCPULNY1mbgUg-dKlgOTiPmhrRuK?usp=drive_link)
- [Weird Tales Collection](https://drive.google.com/drive/folders/1qbw6djNkMjAnHRl37guDhFlebzK9T5Fg?usp=drive_link)
